# Conversor de Monedas en C

## Descripción
Este proyecto consiste en una aplicación desarrollada en el lenguaje de programación **C** que permite convertir entre tres tipos de moneda:

- Quetzales (GTQ)
- Dólares estadounidenses (USD)
- Euros (EUR)

El programa funciona mediante un menú interactivo en la terminal donde el usuario selecciona el tipo de conversión que desea realizar e ingresa la cantidad de dinero.

C es un **lenguaje compilado**, por lo que es necesario **compilar el código antes de ejecutarlo**.

## Prerrequisitos

Para ejecutar este proyecto se necesita:

- GCC (GNU Compiler Collection)
- Sistema operativo Linux, Windows o Mac
- Terminal o consola

Para instalar GCC en Linux (Ubuntu):

```bash
sudo apt update
sudo apt install build-essential
```
